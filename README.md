# 🧮 Calculadora em Java

Uma calculadora desenvolvida em Java utilizando conceitos fundamentais da linguagem.

## Funcionalidades

- ➕ Soma
- ➖ Subtração
- ✖️ Multiplicação
- ➗ Divisão
- 🚫 Tratamento de divisão por zero
- 🔁 Menu interativo
- 📖 Código organizado em métodos

## Tecnologias

- Java
- Scanner
- Switch Expression (Java Moderno)

## Conceitos praticados

- Métodos
- Estruturas de decisão
- Estruturas de repetição
- Organização de código
- Tratamento de erros
- Reutilização de métodos

## Como executar

```bash
git clone https://github.com/SEU-USUARIO/calculadora-java.git
```

Compile:

```bash
javac Main.java
```

Execute:

```bash
java Main
```
